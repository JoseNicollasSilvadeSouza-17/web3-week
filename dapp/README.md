# dapp

Frontend do projeto Webbb3. 

## Setup da Aula

1. Instale Node.js: https://nodejs.org
2. Instale VS Code: https://code.visualstudio.com
3. Biblioteca Bootstrap: https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css
4. Ícone da MetaMask: https://upload.wikimedia.org/wikipedia/commons/thumb/3/36/MetaMask_Fox.svg/1280px-MetaMask_Fox.svg.png?20220831120339
5. Imagem do BBB: https://s2-g1.glbimg.com/oQkh9RvVno32h68-r06gXaPoskI=/0x0:984x554/984x0/smart/filters:strip_icc()/i.s3.glbimg.com/v1/AUTH_59edd422c0c84a879bd37670ae4f538a/internal_photos/bs/2023/q/Q/pAo4MYTy2WU6K4pqNpTw/bbb.jpeg

## Como Executar

1. git clone dapp
2. cd dapp
3. npm install
4. npm run dev
5. acesse localhost:5173 no navegador
